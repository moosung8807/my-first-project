## 추가 후보 슬라이드

- `보통사람 금융생활`: 신한은행 보통사람 금융생활 보고서 기반
- `평균 금융자산`: 하나금융연구소 금융소비자 보고서 기반
- `결혼비용 기준`: 하나금융연구소 금융소비자 보고서 기반
- `노후 안심 총자산`: 하나금융연구소 금융소비자 보고서 기반
- `가계부채 규모`: 한국은행 가계신용 보도자료 기반
- `건강보험 피부양자 기준`: 국민건강보험공단 기준 기반
- `국가건강검진 기준`: 국민건강보험공단 안내문 기반

## 주의

- `금수저 / 흙수저`는 공식 통계 기준이 없으므로 숫자 기준으로 단정하지 않습니다.
- 백분위 자료는 커트라인과 구간 평균을 구분해서 표기합니다.
- 금융기관 리포트는 공공통계가 아니라 설문/자체 분석 자료이므로 `출처`와 `조사 기준`을 함께 표기합니다.
